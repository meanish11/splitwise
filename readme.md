# SplitWise 💸

A lightweight, real-time **group expense splitter** built with vanilla HTML, CSS & JavaScript — no frameworks, no installs.

---

## What it does

You’re on a trip with friends. Different people pay for different things. At the end, nobody knows who owes what. **SplitWise fixes that.**

Add everyone’s name → log who paid for what → hit **Calculate** → instantly see the minimum transfers needed to settle up.

---

## Features

| Feature | Details |
|---|---|
| 👥 People management | Add / remove trip members |
| 💰 Expense tracking | Log payer, amount & description |
| 🧮 Smart settlement | Minimizes number of transfers needed |
| 📊 Breakdown | Per-person paid / fair share / balance |
| ☁️ Firebase sync | Real-time data across devices via group code |
| 📄 PDF export | Download a full settlement report |
| 📱 Responsive | Works on desktop, tablet & mobile |

---

## Tech Stack

- **Frontend** — HTML5 · CSS3 · Vanilla JS (ES6+)
- **Database** — Firebase Firestore (real-time sync)
- **PDF** — jsPDF
- **No build step** — open `index.html` and go

---

## How to Use

1. **Create or Join a group** using the home screen
2. **Add people** in the 👥 People tab
3. **Log expenses** in the 💸 Expenses tab — pick who paid & amount
4. **Hit Calculate** in the 🧾 Settlement tab — see exactly who pays whom
